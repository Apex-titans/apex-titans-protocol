# Free Deploy Guide (Remix VM / Testnet)

## 0) Free local test (no money)
Use Remix VM (Osaka) to test:
1. Deploy OwnershipProofRegistry
2. Deploy ProofVerifier(registry_address)
3. Deploy ATSToken(initialHolder_wallet)
4. Test commit + verify

## 1) If you want real chain (needs gas)
Any real blockchain deployment requires gas (BNB/ETH/etc). Free mainnet deploy does not exist.

## 2) Cheapest path (testnet)
- Use a testnet faucet for free test gas
- Deploy to a testnet first (BSC testnet / Sepolia etc.)
- Verify contracts later when you have small gas

## 3) Notes
- Always start with testnet
- Never send real money to unknown “deploy for free” sites
